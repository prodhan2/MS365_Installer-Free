# MS365_Installer-Free

A simple and easy-to-use tool for installing Microsoft Office.

## Features

- Install Microsoft Office with different editions
- Support for multiple years (2019, 2021, 2024, 2025, Microsoft 365)
- Option to create desktop shortcuts
- Option to exclude Teams
- Option to exclude OneDrive

## How to Use

1. Download or clone the repository
2. Open the solution file in Visual Studio
3. Build the project
4. Run `Office Installer.exe` from the bin directory
5. Select the year and edition
6. Check the options you want
7. Click "Start" to begin installation
